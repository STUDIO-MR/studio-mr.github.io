# $findChannel
Finds a channel's ID from the given channel name, ID, or mention.

## Usage
```
$findChannel[channel]
```

### Breakdown
- `channel` - The channel name/ID/mention to find.

## Example
```
$nomention
$findChannel[$message]
```
