# $modulo
Returns remainder between numbers.

## Usage
```
$modulo[numbers]
```

### Breakdown
- `numbers` - The numbers to return the remainder for. Separate numbers using `;`.

## Example
```
$nomention
$argsCheck[>2;❌ Invalid usage. Usage: `!modulo (number1) (number2)`]
$modulo[$message[1];$message[2]]
```
