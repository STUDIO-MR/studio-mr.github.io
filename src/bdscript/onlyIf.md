# $onlyIf
If *x* is related accordingly (based on the "sign") with *y* then the code runs. If not, the 'errorMessage' is returned.

## Usage
```
$onlyIf[valuex(sign)valuey;errorMessage]
```

### Signs
- `==` - Equal
- `!=` - Not Equal
- `<` - Less Than
- `>`- Greater Than
- `>=` - Greater Than Or Equal To
- `<=` - Less Than Or Equal To
