# $randomText
Picks one value from the provided values randomly.

## Usage
```
$randomText[text1;text2;etc]
```

### Breakdown
- `text1`/`text2`/`etc` - The text which the bot can choose from.

## Example
```
$nomention
$randomText[Hello;Hi;Hey]!
```
