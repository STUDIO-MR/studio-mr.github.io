# $rolePosition
Returns a roles position *(1 being highest role)*.

## Usage
```
$rolePosition[roleID]
```

### Breakdown
- `roleID` - The role to get the position of.

## Example
```
$nomention
$description[<@&$findRole[$message]>'s Position: $rolePosition[$findRole[$message]]]
```
