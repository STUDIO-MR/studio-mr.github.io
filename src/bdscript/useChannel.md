# $useChannel
The bot's response gets sent to the provided channel.

## Usage
```
$useChannel[channelID]
```

### Breakdown
- `channelID` - The channel to redirect the bot's response to.

## Example
```
$nomention
$useChannel[$mentionedChannels[1]]
$description[hi]
$title[hello]
```
