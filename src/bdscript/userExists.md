# $userExists
Returns whether the provided user actually exists.

> 🧙‍♂️ 'true' means the user exists, 'false' means they don't.

## Usage
```
$userExists[userID]
```

### Breakdown
- `userID` - The ID of the hypothetical user.

## Example
```
$nomention
$userExists[$mentioned[1]]
```
